# -Vityarthi-java-project
Campus Course &amp; Records Manager (CCRM) is a Java console app for managing students, courses, grades, and transcripts. It features OOP design, file I/O, GPA calculation, and CSV import/export. Ideal for learning Java architecture and building real-world academic tools.

 Evolution of Java – A Quick Timeline
- 1995 – Java 1.0 released by Sun Microsystems
- 1998 – Java 2 introduced with SE, EE, and ME editions
- 2004 – Java 5 adds generics, annotations, and enhanced for-loops
- 2011 – Oracle takes over; Java 7 introduces try-with-resources
- 2014 – Java 8 brings lambdas and Stream API
- 2017–2023 – Rapid release cycle begins (Java 9 to Java 20+), with modularity, records, pattern matching, and more

 Java Architecture – JDK, JRE, JVM
- JDK (Java Development Kit): Full toolkit for developers. Includes compiler (javac), debugger, and JRE.
- JRE (Java Runtime Environment): Runtime libraries + JVM. Needed to run Java apps.
- JVM (Java Virtual Machine): Executes Java bytecode. Platform-independent engine.


Installing Java on Windows
- Download JDK from Oracle’s site
- Run installer and follow prompts
- Set environment variables:
- JAVA_HOME → path to JDK
- Add %JAVA_HOME%\bin to Path
- Verify with java -version and javac -version in Command Prompt

 Using Eclipse IDE
- Download Eclipse IDE for Java Developers
- Launch Eclipse and create a new workspace
- Go to File → New → Java Project
- Name your project and click Finish
- Right-click src → New → Class → Write your code
- To run: Right-click file → Run As → Java Application
